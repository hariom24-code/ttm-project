# Internship Project — React (Vite)

This is a minimal React app scaffolded with Vite.

Quick start

1. Install dependencies:

   ```powershell
   cd "c:\Users\Hariom kumbhkar\Downloads\internship project"
   npm install
   ```

2. Start dev server:

   ```powershell
   npm run dev
   ```

3. Build for production:

   ```powershell
   npm run build
   npm run preview
   ```

Project files

- `index.html` — app HTML entry
- `package.json` — scripts & deps
- `src/main.jsx` — React entry
- `src/App.jsx` — main component
- `src/index.css` — basic styles

Notes

- This uses React 18 and Vite. If you prefer TypeScript or a different starter, tell me and I can convert it.
